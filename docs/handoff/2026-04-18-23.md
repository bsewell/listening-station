1. **Standalone repo** (not inside GIStudio)
2. **Same Supabase** with `listening_station_` prefix
3. **Qwen 2.5-coder:32b** as default local model (user switched from phi4)
4. **Interviewer Lab** approach: learn from real interviewers, create hybrid persona

6. **Claude API (Sonnet)** for interview generation, **Ollama/Qwen** for briefings and summaries
7. **Multi-format output**: interview transcript, blog post, social clips, audio script with TTS cues
